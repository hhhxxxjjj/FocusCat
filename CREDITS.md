### 像素猫 sprite

- **来源:** "FULL Cat 2D Pixel Art"(付费完整版,作者同 FREE 版)
- **使用文件:**
  - `src/assets/idle.png` ← 原 IDLE.png(8 帧)
  - `src/assets/walk.png` ← 原 WALK.png(12 帧)
  - `src/assets/eat.png` ← 原 EAT.png(7 帧,git commit 触发的"真吃饭")
  - `src/assets/attack.png` ← 原 ATTACK 1.png(8 帧,窗口入侵触发的"扑爪攻击")
  - `src/assets/sleep.png` ← 原 SLEEP.png(8 帧)
- **许可证:** 见 `src/assets/SPRITE_LICENSE.txt`

简述:可用于任何项目(个人 / 商业);不可作为单独的游戏资源转售或再分发;禁止 NFT 化。

资源包内还有 12 个其他状态(Run / Attack 2 / Lick / Scare / Climb / Death / Bowl / Waking up / Slide / Hurt / Jump / Running Jump),Committen v0.1 暂未使用,留作 v0.2+ 弹药库,见 `IDEAS.md`。
